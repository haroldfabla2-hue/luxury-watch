# Plan de Investigación: Servicios de IA para Procesamiento de Imágenes 100% Gratuitos e Ilimitados

## Contexto del Proyecto
**Sistema actual:** Stack con Remove.bg limitado (50 llamadas/mes), COLMAP, OpenCV, etc.
**Objetivo:** Encontrar alternativas completamente gratuitas e ilimitadas para 8 categorías específicas de procesamiento de imágenes.

## Categorías a Investigar
1. **APIs de eliminación de fondo sin límites** (rembg local, alternativas open-source)
2. **Servicios de super-resolución gratuitos sin límites**
3. **APIs de mejora de imágenes completamente gratuitas**
4. **Servicios de generación de texturas PBR gratuitas sin límites**
5. **APIs de normalización de iluminación gratuitas**
6. **Herramientas de clustering y análisis de imágenes open-source**
7. **APIs de análisis de calidad de imágenes (BRISQUE, etc.) gratuitas**
8. **Servicios de generación de metadatos con IA gratuita**

## Criterios de Evaluación
- **Completamente gratuito**: Sin costos de uso
- **Ilimitado**: Sin restricciones de llamadas/mes
- **Viable para producción**: Mínimo nivel de calidad aceptable
- **Implementación local preferida**: Para evitar dependencias externas
- **Open source prioritario**: Para mayor control y customización

- **Rembg**: Solución open-source completa y madura para eliminación de fondo

- **AutoEnhance.ai**: Balance automático de blancos
- **Voxel51**: Guía completa para clustering de imágenes con Scikit-learn
